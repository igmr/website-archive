__Script SQL__, para crear registros hacia la base de datos de __MySQL__.

## 🛬 Importar script desde terminal

Para Importar un ___archivo___ de __script SQL__, ejecutamos el siguiente comando.

```bash
# Sintaxis
mysql -u <user_name> -p < <path/script/sql>

#Ejemplo
mysql -u root -p < ~/inserts.sql
```
