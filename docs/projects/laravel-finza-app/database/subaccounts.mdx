Para tener mayor control de nuestro efectivo, los bancos
brindan la posibilidad de administrar nuestro efectivo en
__sub cuentas__, aunque cada banco lo llama de diferente manera,
pero el objetivo es el mismo. Esta tabla representará esta
separación de efectivo en __sub cuentas__.
