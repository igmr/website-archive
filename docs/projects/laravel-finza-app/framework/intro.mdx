__Laravel__ es un framework __PHP__, para aplicaciones web, usando el patrón de diseño __MVC__.

## 🗄️ Base de datos soportados

Como casi todas las aplicaciones web modernas interactúan con una bases de datos,
__Laravel__ permite la integración extremadamente simple, ofreciendo la utilización de
del lenguaje __SQL__, un generador de consulta y el __ORM Eloquent__.

Actualmente laravel solamente soporta estos 5 gestores de bases de datos.

<Ul>
    <li>🟢 MariaDB 10.3+</li>
    <li>🟢 MySQL 5.7+</li>
    <li>🟢 PostgreSQL 10.0+</li>
    <li>🟢 SQLite 3.8.8+</li>
    <li>🟢 SQL Server 2017+</li>
</Ul>


## ⚒️ Requisitos técnicos del servidor

Para subir el proyecto de __Laravel__ a ___producción___, se  requiere de algunos requerimientos mínimos para el funcionamiento correcto.
Asegúrese de tener los siguientes requerimientos:

<Ul>
    <li>✅ PHP >= 8.1.</li>
    <li>✅ Ctype PHP Extension.</li>
    <li>✅ cURL PHP Extension.</li>
    <li>✅ DOM PHP Extension.</li>
    <li>✅ Fileinfo PHP Extension.</li>
    <li>✅ Filter PHP Extension.</li>
    <li>✅ Hash PHP Extension.</li>
    <li>✅ Mbstring PHP Extension.</li>
    <li>✅ OpenSSL PHP Extension.</li>
    <li>✅ PCRE PHP Extension.</li>
    <li>✅ PDO PHP Extension.</li>
    <li>✅ Session PHP Extension.</li>
    <li>✅ Tokenizer PHP Extension.</li>
    <li>✅ XML PHP Extension.</li>
</Ul>
